Map description
========

The map is encoded as a list of lists (row-major) representing the 2-d
grid. An enumeration represents the contents of each grid cell:
* 0: Wall (`#`)
* 1: Empty (`<space>`)
* 2: Pill (`.`)
* 3: Power pill (`o`)
* 4: Fruit location (`%`)
* 5: Lambda-Man starting position ('\')
* 6: Ghost starting position ('=')
